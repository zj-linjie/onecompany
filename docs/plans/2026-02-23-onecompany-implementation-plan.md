# OneCompany v0.1 Implementation Plan

Date: 2026-02-23

## Milestones

1. Monorepo scaffolding and TypeScript build chain
2. Core modules: templates, scanner, skill router, flow runners
3. Interactive CLI with three paths: new project, takeover, iterate
4. Validation: build/typecheck and smoke run

## Deliverables

- Interactive CLI command flow
- Workspace/docs generation pipeline
- Existing project scanner with findings and risks
- Task-to-skill routing and quality gate checklist

## Out of scope (v0.1)

- Git URL and Notion direct import
- Web UI dashboard
- multi-agent runtime parallelism engine
